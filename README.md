# Business Market Analytics Dashboard

## Project Overview

This project is an end-to-end business analytics solution developed using Python, MySQL, and Power BI. The project analyzes coworking spaces and marketing agencies to uncover customer engagement patterns, category performance, business ratings, and market trends.

The workflow includes data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and interactive dashboard development.

---

## Tools & Technologies

- Python
  - Pandas
  - NumPy
- MySQL
- Power BI
- Power Query
- DAX

---

## Project Workflow

### 1. Data Collection

The project uses business listing datasets containing:

- Business Name
- Ratings
- Reviews
- Main Category
- Operating Hours
- Competitor Information
- Business Metadata

---

### 2. Data Cleaning & Preparation (Python)

Tasks performed:

- Removed duplicate records
- Handled missing values
- Standardized categorical values
- Cleaned operating hour information
- Removed irrelevant fields
- Created analytical features
- Performed data validation checks

---

### 3. Exploratory Data Analysis (EDA)

Key analyses performed:

- Rating distribution analysis
- Review distribution analysis
- Category frequency analysis
- Operating hour analysis
- Competitor analysis
- Customer engagement analysis

---

### 4. SQL Analysis (MySQL)

Business questions answered:

- Total businesses by category
- Average ratings across categories
- Top-performing businesses by reviews
- Customer engagement patterns
- Category-level market analysis
- KPI generation for dashboard reporting

---

### 5. Dashboard Development (Power BI)

Two interactive dashboard pages were developed.

---

## Dashboard 1: Coworking Market Overview

### KPIs

- Total Locations
- Rated Locations
- Average Rating
- Total Reviews
- 24-Hour Locations

### Visuals

- Top 5 Business Categories
- Workday Type Distribution
- Average Rating by Category
- Top Reviewed Coworking Spaces

### Key Insights

- Coworking spaces dominate the market.
- Average ratings remain consistently high.
- Most businesses operate fixed hours.
- Customer engagement is concentrated among a few leading workspaces.

---

## Dashboard 2: Marketing Agency Market Overview

### KPIs

- Total Agencies
- Rated Agencies
- Average Rating
- Total Reviews

### Visuals

- Top Marketing Agencies by Reviews
- Top Agency Categories
- Agency Rating Distribution
- Customer Engagement by Category

### Key Insights

- Marketing agencies represent the largest category.
- Average customer satisfaction exceeds 4.7.
- Most agencies receive highly positive ratings.
- Customer engagement is concentrated within a small number of categories.
